# MFC-1 project

    Установка и запуск:

1. Должен быть установлен и запущен Firebird сервер (я ставил 2.5.4, super).
2. Нужно скопировать файлы из папки 'application' в отдельную директорию (2 файла).
3. Файл БД - это пустая база с требуемой структурой.
4. Запустить исполняемый файл.
5. Другие дополнительные настройки не требуются.


    Комментарии:

- IDE: MS Visual Studio pro 2010 SP1.
- приложение MFC.
- в папке "VS project archive" - архив проекта VS.
- логин и пароль к БД - стандартные: SYSDBA/masterkey.
- IBPP - исходники интегрированы в проект (v.2.5.3.1).
- при операции "Выдать всем" будет сформирован файл отчета report.txt - его можно просмотреть нажав кнопку "Отчеты" (было удобно при тестировании, я не стал её удалять).
